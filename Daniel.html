A good number of the bus tracking systems features were if the user clicked on the website and see bus timetables. There was a front end and a back end. The back end was updated manually so each time the bus stopped, it was updated and then posted to the front end.

Challenges faced:

• To find a solution for users to see a bus system and brainstormed ideas and asked the supervisor for ideas.

• Deciding what type of bus location system to use by analyzing different bus tracking systems like Dublin Bus and other bus companies.

• Determining how the system would work the next step was brainstorming and doing scrum sessions.
